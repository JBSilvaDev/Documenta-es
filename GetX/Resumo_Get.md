- Instalação:
    ```dart
        flutter pub add get
    ```
    ```yaml
        dependencies:
            get: ^4.6.5
    ```
- Disponibilizar Get para aplicação
    - Substituir o `MaterialApp()` por `GetMaterialApp()`
